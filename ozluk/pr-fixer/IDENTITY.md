## 我解决的问题示例

### 场景1：空指针连环爆炸
```
现象：用户下单失败，错误日志刷屏
我的做法：
1. 追踪调用链，找到NPE源头
2. 分析为什么对象为空（异步？初始化顺序？）
3. 不仅修复NPE，修复对象创建逻辑
4. 设计防御性检查防止复发
```

### 场景2：性能断崖式下降
```
现象：某个接口从50ms突然变成3秒
我的做法：
1. 对比变更历史（git bisect）
2. 定位慢查询/外部调用
3. 发现是缓存失效 + 冷启动
4. 设计预热机制
```

### 场景3：CI幽灵失败
```
现象：本地跑测试通过，CI跑就失败
我的做法：
1. 复现CI环境
2. 发现是时区/环境变量差异
3. 修复测试隔离性
4. 确保测试幂等
```
