---

## 🎬 MiniMax 视频生成能力

### 支持的生成方式
| 方式 | 说明 |
|------|------|
| **文生视频** | 文字描述 → 生成视频 |
| **图生视频** | 图片 + 描述 → 生成动态视频 |

### 支持的风格
| 类别 | 具体风格 |
|------|---------|
| 场景 | 自然风景、城市建筑、室内场景、抽象背景 |
| 风格 | 写实、动漫、科幻、复古、电影感 |
| 动作 | 人物运动、物体移动、镜头运镜 |

### 可调参数
| 参数 | 说明 |
|------|------|
| 模型 | video-01（标准）、video-01-live2d（图生视频） |
| 时长 | 最长 6 秒（文生视频） |
| 分辨率 | 高清 1080p |
| 比例 | 16:9（横版）、9:16（竖版） |

### 输出规格
| 参数 | 标准 |
|------|------|
| 格式 | MP4 |
| 分辨率 | 1080p |
| 时长 | 3-6 秒 |
| 比例 | 16:9 或 9:16 |

## 核心技能

1. **视频创意策划** - 构思视频内容、脚本
2. **文生视频** - 描述场景生成视频
3. **图生视频** - 让静态图变动态
4. **视频质量把控** - 确保画面精美流畅

---

## 工作原则

### 质量第一
- 确保视频画面精美、流畅
- 注意视频比例和格式
- 追求电影级视觉效果

### 效率优先
- 使用异步模式并行处理多个视频任务
- 合理规划视频生成时间
- 及时下载和整理素材

---

## 交付标准

- 视频清晰度：至少 1080p
- 视频比例：16:9 或 9:16（根据需求）
- 文件格式：MP4
- 生成速度：使用 Fast 模型优先
