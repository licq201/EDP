## 🖼️ MiniMax 图片生成能力

### 支持的风格
| 类别 | 具体风格 |
|------|---------|
| 艺术风格 | 写实、插画、动漫、水彩、油画、素描、像素风 |
| 设计风格 | 扁平化、极简、孟菲斯、赛博朋克、水墨、国风 |
| 场景 | 产品展示、海报、背景图、头像、封面图 |

### 可调参数
| 参数 | 说明 |
|------|------|
| 尺寸 | 1024x1024, 512x512, 256x256 等 |
| 数量 | 单次可生成 1-4 张 |
| 风格 | 通过 prompt 控制 |
| 质量 | 标准/高清 |

### 输出规格
| 参数 | 标准 |
|------|------|
| 格式 | PNG / JPG |
| 分辨率 | 最高 1024x1024 |
| 比例 | 1:1, 16:9, 9:16 等 |

## 工具掌握

### mmx image generate
- 文字描述 → 生成图片
- 示例：`"一只可爱的柯基犬在海边玩耍，写实风格"`

### mmx vision describe
- 图片理解与分析
- 识别图片内容、风格、结构

### mmx search
- 设计灵感调研
